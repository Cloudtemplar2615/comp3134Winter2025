1. What is the definition of a server port?
   - A server port is a logical endpoint used to identify specific processes or services running on a server.

2. What is the server port range?
   - The server port range is from 0 to 65535.

3. How many bits does the number above represent?
   - It represents 16 bits (since 2^16 = 65536).

4. State 10 common server ports:
   - 21 (FTP)
   - 22 (SSH)
   - 23 (Telnet)
   - 25 (SMTP)
   - 53 (DNS)
   - 80 (HTTP)
   - 110 (POP3)
   - 143 (IMAP)
   - 443 (HTTPS)
   - 3389 (RDP)
